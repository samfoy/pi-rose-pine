# pi-rose-pine

[Rosé Pine](https://rosepinetheme.com/) theme for [pi](https://shittycodingagent.ai).

## Install

```bash
pi install npm:pi-rose-pine
```

Then select it via `/settings` → theme → `rose-pine`.

## Colors

Based on the Rosé Pine palette — muted, warm tones with just enough contrast for long sessions.

| Role | Color |
|------|-------|
| Accent | Iris `#c4a7e7` |
| Success | Foam `#9ccfd8` |
| Error | Love `#eb6f92` |
| Warning | Gold `#f6c177` |
| Headings | Rose `#ebbcba` |
| Code | Foam `#9ccfd8` |
